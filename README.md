# Dog Breed Classification: Custom CNN vs ResNet18

Этот проект посвящен классификации пород собак (на 5 классов) по изображениям с использованием библиотеки **PyTorch**. В репозитории сравниваются два подхода: обучение собственной сверточной нейросети (Custom CNN) с нуля и Transfer Learning на базе предобученной модели **ResNet18**.

В конце скрипта также реализован экспорт лучшей модели в формат **TorchScript** для оптимизации и дальнейшего деплоя.

---

## 🚀 Основные возможности
* **Препроцессинг данных**: Изменение размера до 224x224, конвертация в тензоры и разделение датасета на обучающую и тестовую выборки (80/20).
* **Собственная архитектура (Custom CNN)**: Сверточная сеть с использованием слоев `BatchNorm2d`, `MaxPool2d` и `Dropout2d`.
* **Transfer Learning**: Замораживание весов базовой архитектуры `ResNet18` и обучение только финального полносвязного слоя (`fc`) под 5 классов.
* **Анализ результатов**: Оценка качества с помощью метрик `classification_report`, матрицы ошибок (`confusion_matrix` визуализированная через Seaborn) и графика истории потерь (Loss History).
* **Production-ready экспорт**: Сохранение весов (`.pth`) и трассировка модели через `torch.jit.trace` (`.pt`).

---

## 📂 Структура данных

Для работы скрипта необходимо подготовить датасет в формате `ImageFolder`. Создайте папку `dog_dataset` в корневой директории проекта:

```text
dog_dataset/
├── breed_1/
│   ├── img1.jpg
│   └── img2.jpg
├── breed_2/
│   └── ...
└── ... (всего 5 папок для 5 пород) ```
